Chapter 01

Luồng Khởi Động

API Startup Sequence

Tầng 1 — Configuration Binding

Load .env → bind JwtSettings (Secret, Issuer, Audience, ExpiryMinutes=1440), SUPABASE_CONNECTION_STRING, SUPABASE_URL, SUPABASE_SERVICE_ROLE_KEY.

Tầng 2 — EF Core Migration Auto-Apply

Gọi db.Database.MigrateAsync() khi khởi động. Hỗ trợ cả PostgreSQL URI format lẫn key-value connection string. Nếu lỗi → log warning, không crash server.

Tầng 3 — Data Seeding (SeedService)

Kiểm tra DB trống → seed Admin account (Role=1), sample POIs, default content. Chỉ chạy nếu Users.Count() == 0.

Tầng 4 — JWT Middleware

Đăng ký AddAuthentication(JwtBearer) với TokenValidationParameters. Đọc cả Authorization: Bearer header lẫn cookie AuthToken.

Server Ready — 5 Controllers Mounted

AuthController, POIController, AnalyticsController, UserController, ReportController → prefix /api/.

Web Startup Flow

App Start → Cookie Auth Middleware → Check AuthToken Cookie → Decode JWT Role → Route by Role

🔑 Role=1 (Admin)

Redirect → /Home/Dashboard. Sidebar layout với full CRUD POI, analytics, user management.

🏪 Role=2 (ShopOwner)

Redirect → /ShopOwner/Dashboard. Chỉ thấy POI của mình, view statistics, edit content.

📱 Mobile Startup

Kiểm tra SecureStorage["jwt_token"] → nếu có và còn hạn → vào MainPage, ngược lại → AuthPage.

Mobile Startup Sequence

JWT Check + Auto-Login

MobileAuthService.IsAuthenticated() → decode token, check exp field (số, không phải string). Dùng JsonValueKind.Number guard.

SQLite Sync (POIService)

Load pois.db từ local → hiển thị ngay (0ms offline). Sau đó gọi GET /api/poi → sync về SQLite → cập nhật UI.

GPS Permission Request

Android: xin ACCESS_FINE_LOCATION + ACCESS_BACKGROUND_LOCATION → khởi LocationForegroundService. iOS: WhenInUseAuthorization.

GeofenceService Activation

Subscribe LocationService.LocationChanged → bắt đầu monitor tất cả active POI trong cache. Cooldown 5 phút per POI.

JWT Implementation

🔐 Token Generation (API)

  • Algorithm: HS256
  • Secret: 32-byte key (env var)
  • Expiry: 1440 phút (24h)
  • Claims: NameIdentifier, Name, Email, Role
  • BCrypt password hash (cost factor 12)

🌐 Web Cookie Auth

  • Cookie name: AuthToken
  • Đọc JWT → parse claims thủ công
  • Tạo ClaimsPrincipal → SignInAsync()
  • Logout: SignOutAsync() + xóa cookie
  • Redirect về /Auth/Login nếu 401

📱 Mobile SecureStorage

  • Lưu JWT vào SecureStorage["jwt_token"]
  • Android: Encrypted SharedPreferences
  • iOS: Keychain Services
  • Đính kèm Bearer header mỗi API call
  • Auto-retry với refresh nếu 401

JWT Claims Parsing — Known Pitfall

// JWT claims dùng full URI namespace: var role = claims["http://schemas.microsoft.com/ws/2008/06/identity/claims/role"]; // exp field là Number, không phải String — phải guard: if (prop.Value.ValueKind != JsonValueKind.String) continue; // GetClaimsFromJwt xử lý cả URI và shorthand: private static Dictionary<string, string> GetClaimsFromJwt(string token) { var payload = token.Split('.')[1]; var json = Base64UrlDecode(payload); // parse → return dict bỏ qua Number fields }

Login Flow

POST /auth/login → Verify BCrypt → Generate JWT → Return {token, role} → Store Cookie / SecureStorage

Chapter 03

GPS + Geofence + Tự Động Phát Thuyết Minh

Khi khách du lịch bước vào vùng bán kính của POI (mặc định 50m), hệ thống tự động phát âm thanh thuyết minh bằng ngôn ngữ đã chọn, không cần thao tác thêm.

Kiến Trúc GPS Trên Mobile

LocationService — GPS Continuous Tracking

Geolocation.StartListeningForegroundAsync() → emit LocationChanged event liên tục. Android dùng LocationForegroundService để track khi app background.

GeofenceService — Zone Detection

Subscribe LocationChanged → tính Haversine distance với mỗi POI trong cache → nếu distance ≤ poi.Radius → trigger narration. Cooldown 5 phút per POI (tránh phát lặp).

NarrationService — Queue Manager

Nhận POI trigger → enqueue vào NarrationQueue (single-slot priority). Xử lý tuần tự: không phát đè lên audio đang chạy. Hỗ trợ Skip/Stop.

ITextToSpeech — Platform TTS

Đọc POIContent theo ngôn ngữ user đã chọn (vi/en/zh) → gọi TextToSpeech.SpeakAsync(text, options). Options gồm volume, pitch, locale.

VisitLog — Fire-and-Forget Analytics

Song song với phát audio: POST /api/analytics/visit với VisitType=Geofence, lat/lng, UserId, timestamp. Không block UI.

Visit Type Enum

Geofence
VisitType = 1 — Tự động khi đến gần Trigger bởi GeofenceService khi user bước vào vùng bán kính POI. Phổ biến nhất.
MapClick
VisitType = 2 — User chủ động click trên bản đồ Tap vào POI marker trên Leaflet map → hiện detail + phát thuyết minh thủ công.
QRCode
VisitType = 3 — Quét mã QR tại điểm Quét QR code tại cửa hàng → mở POI detail page → log visit.
Android Background Location: Từ Android 10+, cần xin riêng quyền ACCESS_BACKGROUND_LOCATION. App hiển thị dialog giải thích trước khi request. Nếu từ chối, geofence vẫn hoạt động khi app foreground.

Chapter 05

Hệ Thống Thuyết Minh Audio (TTS)

Mobile sử dụng .NET MAUI built-in ITextToSpeech — không cần API key, không phát sinh chi phí. Hỗ trợ các giọng đọc platform-native trên Android và iOS.

NarrationService — Kiến Trúc Queue

// NarrationService.cs — single-slot priority queue public class NarrationService { private CancellationTokenSource _cts; private readonly Queue<POI> _queue = new(); public async Task EnqueueAsync(POI poi, string language) { _cts?.Cancel(); // stop current narration _cts = new CancellationTokenSource(); var content = poi.Contents.FirstOrDefault(c => c.Language == language) ?? poi.Contents.FirstOrDefault(c => c.Language == "vi"); if (content == null) return; await TextToSpeech.SpeakAsync(content.TextContent, new SpeechOptions { Locale = GetLocale(language), Volume = 1.0f }, _cts.Token); } }

Voice Settings (VoicePreferenceService)

🇻🇳 Tiếng Việt

Locale: vi-VN. Giọng platform: Android chọn Google TTS Vietnamese, iOS chọn Siri Linh.

🇬🇧 English

Locale: en-US. Giọng platform: Android Google TTS US English, iOS Samantha.

🇨🇳 中文

Locale: zh-CN. Giọng platform: Android Google TTS Mandarin, iOS Tingting.

Luồng Phát Thuyết Minh Đầy Đủ

Geofence Trigger → Lookup POIContent (lang) → Language Fallback → NarrationService.Enqueue → ITextToSpeech.SpeakAsync
AudioUrl Priority: Nếu POIContent.AudioUrl có giá trị (pre-recorded file), Mobile phát file âm thanh thay vì TTS. Chất lượng cao hơn, không phụ thuộc platform voice.

TourGeneratorService

Tạo tự động hành trình tham quan dựa trên vị trí hiện tại và danh sách POI. Sắp xếp theo độ ưu tiên và khoảng cách → tạo Tour với ordered POI list → phát thuyết minh theo thứ tự khi đi.

Analytics & Báo Cáo

Nguồn số liệu chính là bảng VisitLogs: mỗi lượt ghé ghi một dòng. Dashboard & báo cáo được tính trực tiếp từ VisitLogs qua IAnalyticsService (GROUP BY, COUNT, filter theo ngày/POI) — không dùng bảng tổng hợp Analytics. n8n có thể gọi API báo cáo theo khoảng thời gian.

📝 VisitLog (sự kiện)

  • Ghi mỗi lần khách ghé thăm POI
  • Fields: POIId, UserId, VisitTime, Lat/Lng, VisitType, DurationSeconds
  • UserId nullable (anonymous tracking)
  • Fire-and-forget từ Mobile → không block UX

📈 Thống kê (runtime)

  • Summary, top POI, thống kê theo POI: truy vấn VisitLogs (EF Core / LINQ)
  • Không có bảng “daily aggregate” trong luồng hiện tại
  • Phân loại theo VisitType (Geofence, MapClick, QRCode) trên tập log

🤖 Daily Report API

  • GET /api/report/daily?date=&token=
  • Bảo vệ bởi REPORT_TOKEN header
  • Trả JSON tổng hợp cho n8n
  • n8n webhook → email/Slack/certificate

👣 User Daily Visits

  • Đếm số POI unique mỗi user ghé trong ngày
  • Dùng để cấp chứng nhận du lịch (n8n)
  • Endpoint: GET /api/report/user-visits
  • PostgreSQL: load → LINQ-to-Objects (tránh COUNT DISTINCT *)

Admin Analytics Dashboard — Các Metrics

MetricCách tínhHiển thị
Total Visits (Today) COUNT VisitLogs WHERE date = today Số lớn trên dashboard
Geofence % COUNT VisitType=1 / Total * 100 Progress bar
Top 5 POIs GROUP BY POIId, ORDER BY COUNT DESC, TAKE 5 Bar chart
Hourly Distribution GROUP BY HOUR(VisitTime) Line chart
Growth % (Today - Yesterday) / Yesterday * 100 Badge up/down
ShopOwner Stats WHERE POI.OwnerId = currentUser Filtered dashboard
PostgreSQL Pitfall: COUNT(DISTINCT *) không được hỗ trợ trong PostgreSQL. Giải pháp: load data vào memory trước bằng .ToListAsync(), sau đó dùng LINQ-to-Objects .GroupBy().Count().

Tour gợi ý & SQLite trên Mobile

Tour — “Bắt đầu Tour”

Trên MainPage, người dùng chọn một tour (ví dụ ẩm thực đường phố) → điều hướng TourDetailPage (route đăng ký Shell). Màn hình hiển thị danh sách POI trong tour, tổng thời gian, khoảng giá ước tính.

Chọn tour → TourDetailPage → Bắt đầu Tour → MapPage

🎯 TourSelectionService (Singleton)

Thuộc tính SelectedTour lưu tour hiện tại trong bộ nhớ (không persist SQLite). Khi bấm Bắt đầu Tour, ViewModel gán SelectedTour = Tour rồi Shell.Current.GoToAsync("//MapPage").

🗺️ Lọc POI trên bản đồ

MapPageViewModel.LoadPOIsAsync(): nếu SelectedTour có danh sách POIs → chỉ hiển thị & khởi tạo geofence cho các POI đó; ngược lại dùng toàn bộ POI từ SQLite (sync từ API).

⚠️ Lưu ý triển khai

Nếu user đã mở tab Bản đồ trước khi bấm Bắt đầu tour, LoadPOIsAsync có thể đã chạy khi SelectedTour còn null — bản đồ vẫn hiện tất cả POI cho đến khi có bước reload (có thể cải tiến: gọi lại load khi OnAppearing Map).

SQLite — pois.db

Chỉ trên Mobile: Web/API dùng PostgreSQL. App dùng sqlite-net-pcl + SQLitePCLRaw.bundle_e_sqlite3. File: FileSystem.AppDataDirectory/pois.db.

POIContentNội dung theo ngôn ngữ, liên kết POId

Đồng bộ: POIService.SyncPOIsFromServerAsync() gọi GET /api/poi → nếu response null hoặc rỗng thì giữ nguyên cache local; nếu có dữ liệu thì xóa bảng và insert lại (full replace).
